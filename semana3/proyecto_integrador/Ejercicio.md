# SISTEMA DE GESTIÓN DE BIBLIOTECA

## Requisitos:
1. Clases: Libro, Usuario, Biblioteca
2. Funcionalidades:
   - Agregar libros
   - Registrar usuarios
   - Prestar libros (máximo 3 por usuario)
   - Devolver libros
   - Buscar libros (por título o autor)
   - Ver libros prestados por usuario
   - Multa por libros atrasados
3. Persistencia:
   - Guardar libros en libros.json
   - Guardar usuarios en usuarios.json
   - Guardar préstamos en prestamos.json
4. Menú interactivo completo

Tiempo estimado: 6-8 horas - 
Dividir en sesiones de 2 horas


### Este proyecto NO debe usar IA
### Si te atoras, divide el problema más
### Ejemplo: Primero solo agregar/listar libros
### Luego agregar usuarios
### Luego préstamos, etc.
